# 🌐 Pexelicons Website

## 📌 Project Description

This is a modern and responsive frontend website built using HTML, CSS, and JavaScript.
The project focuses on clean UI design and smooth user experience.

## 🚀 Features

* Responsive design (Mobile + Desktop)
* Clean and modern UI
* Easy navigation
* Fast loading pages
* Image and icon integration

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

* index.html → Main homepage
* style.css → Styling file
* script.js → JavaScript functionality
* images/ → All images used in project


## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
